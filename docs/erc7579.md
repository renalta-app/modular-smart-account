
### Empty Return Data Signaling

When using `executeFromExecutor()` with EXECTYPE_TRY, failed executions return an empty array:

```solidity
// Executor module pattern
bytes[] memory results = account.executeFromExecutor(mode, executionCalldata);

if (results.length == 0) {
    // Execution failed (only possible with EXECTYPE_TRY)
    // Handle gracefully without reverting
} else {
    // Execution succeeded
    // results[0] contains return data (may be empty bytes for void functions)
}
```

**Key distinction:**
- `results.length == 0` → execution failed
- `results.length == 1 && results[0].length == 0` → execution succeeded with no return data
